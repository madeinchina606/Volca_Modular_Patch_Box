# Volca Modular Patch Box

A single-file web app for drawing and saving patches for the **Korg Volca Modular**. Drag cables, twist knobs, jot down notes, and come back tomorrow without trying to remember how you got that sound.

## Features

- Recreation of the patching area, with four themes (device, dark, B&W, paper diagram)
- Drag-cable patching with snap-to-jack (or click-patching if you prefer)
- Signal-flow colour coding  audio red, CV blue, gates green
- Save patches by name with notes; export/import as JSON
- Print or export as PNG (one per patch, or all stacked into one)

## Usage

1. Download `volca_patch_designer.html`
2. Open it in any modern browser
3. Patch away

No install, no server, no dependencies. Patches live in your browser's `localStorage` — use **Export JSON** if you want a real backup.
