Audio CV Gate Mixed
Drag patching — drag from any jack to another; drop on empty paper to cancel.
Click patching — two-click mode (toggle in header).
Drag Cable (or hold CTRL) — grab a connected jack to re-route; off → cables locked, multiple per jack allowed.
Delete cable — click its body.
Signal-flow colours — when on, cables are tinted by signal type; off → random colour.
Save patches by name — same name overwrites, new name creates a new one. Patches live in local storage; use Export JSON for backups.
Steps — click any step to toggle on/off; the highlighted-yellow step plays at the cursor.
SPACE — start / stop the sequencer (also wakes up the audio engine).
← → — move the cursor between steps.
↑ ↓ — pitch the current step up / down within the active scale (with current microtuning), plays a preview.
− — toggle current step on/off (captures the most recent note).
= — insert a rest (silent step, advance cursor).
` or REC button — live-record: any key you play fills the current step and advances.
Edit a step — click it to highlight, then press a key to set its note (or use ↑ ↓).
−16 / +16 — shrink or grow the pattern by 16. Len input — type any length 1–64. Tempo follows the CLOCK knob.
A W S E D F T G Y H U J K… — play notes (QWERTY piano, white & black keys).
Z / X — octave down / up.
3–9 — Major, Dorian, Phrygian, Lydian, Mixolydian, Minor, Locrian. C — Chromatic.
1 — toggle Long-envelope mode (×3 attack/release).
2 — randomize per-note microtuning. Shift+2 — clear microtuning.
0 — panic: silence modulation.
